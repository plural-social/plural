# Server Profile

A _profile_ on this server describes a page with a set username.

Profiles are typically one of the following:

- A page for a single individual
- A page for an organization/project/buisness/team,
  that one or more individuals can access
- A shared system account for multiple [identities][identity]
  (for [plural systems][plural-system])

Each profile designates one or more [identities][identity] that can post content
and/or manage the profile.
